📚 How It Works

🎯 Automatic Calculation

As you enter coordinates, the drive time is automatically calculated using the haversine formula. This calculates straight-line distance and converts to estimated drive time based on typical speeds for different distance ranges.

⚙️ Speed Assumptions

• Urban: 25 mph (0-5 miles)
• Suburban: 35 mph (5-15 miles)
• Mixed: 35 mph (15-50 miles)
• Highway: 55 mph (50+ miles)

💡 Tips

  • The calculation is an estimate. Actual drive time depends on traffic, route, and conditions.
  • Use Google Maps for more accurate calculations (requires API key).
  • Results are cached for 1 hour to improve performance.
  • Works offline using haversine formula.

🚀 How to Find Coordinates

  1. Go to Google Maps (maps.google.com)
  2. Search for the location
  3. Right-click on the location pin
  4. Click "What's here?" at the bottom
  5. Copy the coordinates shown
