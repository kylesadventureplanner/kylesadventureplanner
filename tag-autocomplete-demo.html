📚 How to Use

ℹ️ Getting Started: The tag input is embedded above. Start typing to see autocomplete suggestions powered by full-text search with fuzzy matching.

Step 1: Include Required Scripts 

// In your HTML head:
<script src="/JS Files/consolidated-tag-system-v7-0-141.js"></script>
<script src="/tag-autocomplete-input.js"></script>

Step 2: Add HTML Container 

// Where you want the tag input:
<div id="tagAutocompleteContainer"></div>

Step 3: Initialize with JavaScript 

// In your script:
document.getElementById('tagAutocompleteContainer').innerHTML = TAG_AUTOCOMPLETE_HTML;

const tagInput = initTagAutocomplete({
  maxTags: 10,
  minSearchLength: 1,
  showConflictWarnings: true,
  onSelectionChange: (tags) => {
    console.log('Selected tags:', tags);
  }
});

Step 4: Get Selected Tags 

// When saving your form:
const selectedTags = tagInput.getSelectedTags();
console.log('Tags to save:', selectedTags);

// Example usage in form submission
form.addEventListener('submit', (e) => {
  e.preventDefault();
  const tags = tagInput.getSelectedTags();
  // Save with your data...
});

⚙️ Configuration Options

Option Type Default Description
maxTags number 10 Maximum number of tags allowed
minSearchLength number 1 Minimum characters before showing suggestions
suggestionLimit number 10 Max suggestions to display
debounceDelay number 150 Delay (ms) before searching
showConflictWarnings boolean true Show conflict warnings
onSelectionChange function null Callback when tags change

🎯 Available Methods

Get Selected Tags 

tagInput.getSelectedTags()
// Returns: ['Hiking', 'Photography-Worthy', 'Family-Friendly']

Set Selected Tags 

tagInput.setSelectedTags(['Easy', 'Scenic'])
// Programmatically set tags

Clear All Tags 

tagInput.clearTags()
// Remove all selected tags

✨ Features

🔍 Smart Autocomplete

5 matching strategies: Exact (100%), Prefix (90%), Substring (70%), Description (50%), and Fuzzy (0-100%) for typo tolerance.

⚠️ Conflict Detection

Automatically detects conflicting tags like "Easy" + "Challenging" and prevents invalid combinations.

🏷️ Tag Aliases

Type "Trekking" and it normalizes to "Hiking". Handles user input variations automatically.

🎨 Beautiful UI

Smooth animations, responsive design, dark mode ready, and fully customizable styling.

💡 Example: Complete Integration

// Complete working example:

document.addEventListener('DOMContentLoaded', () => {
  // Setup tag input
  const container = document.getElementById('tagAutocompleteContainer');
  container.innerHTML = TAG_AUTOCOMPLETE_HTML;

  const tagInput = initTagAutocomplete({
    maxTags: 15,
    onSelectionChange: (tags) => {
      // Update live preview
      document.getElementById('tagPreview').textContent =
        tags.join(', ') || 'No tags selected';
    }
  });

  // Handle form submission
  document.getElementById('locationForm').addEventListener('submit', async (e) => {
    e.preventDefault();

    const location = {
      name: document.getElementById('locationName').value,
      description: document.getElementById('locationDescription').value,
      tags: tagInput.getSelectedTags()
    };

    console.log('Saving location:', location);

    // TODO: Send to your backend
    // await fetch('/api/locations', {
    //   method: 'POST',
    //   body: JSON.stringify(location)
    // });

    // Show success message
    document.getElementById('successMessage').style.display = 'block';
    setTimeout(() => {
      document.getElementById('successMessage').style.display = 'none';
    }, 3000);
  });
});

💡 Tips & Tricks

🎯 Tip 1: Use the onSelectionChange callback to update other parts of your UI in real-time.
🎯 Tip 2: Type partial tag names like "fam" to see all family-related tags.
🎯 Tip 3: Conflicts are shown immediately - users can't make mistakes!
🎯 Tip 4: Tag aliases mean "Trekking" or "Walking" all become "Hiking" for consistency.
🎯 Tip 5: Use tagInput.getSelectedTags() in form submit handlers to save tags with your data.
