## 🔧 API METHODS (Simple & Powerful)

```javascript
// Get tags
const tags = tagInput.getSelectedTags();

// Set tags programmatically
tagInput.setSelectedTags(['Hiking', 'Easy']);

// Add single tag
tagInput.addTag('Photography-Worthy');

// Remove single tag
tagInput.removeTag('Hiking');

// Clear all tags
tagInput.clearTags();

// Respond to changes
tagInput = initTagAutocomplete({
  onSelectionChange: (tags) => console.log(tags)
});
```

## ⚙️ CONFIGURATION OPTIONS

```javascript
initTagAutocomplete({
  maxTags: 10,                    // Max number of tags
  minSearchLength: 1,             // Min chars to search
  suggestionLimit: 10,            // Max suggestions shown
  debounceDelay: 150,             // Search delay (ms)
  showConflictWarnings: true,     // Detect conflicts
  onSelectionChange: (tags) => {} // Callback function
});
```
