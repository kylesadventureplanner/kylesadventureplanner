# Recipes & Shopping List System - Quick Start Guide

## Where to Find It

**Household Tools Tab** → Choose your task:
- 📋 **Recipes** - Create and manage recipes
- 🛒 **Shopping Lists** - Generate shopping lists from recipes
- 🧂 **Pantry** - Track your pantry inventory with filters
- 📦 **Ingredients Library** - Browse 170+ pre-saved ingredients

## Quick Tasks

### ➕ Create a New Recipe (5 minutes)

1. Click **Household Tools** tab
2. Select **Recipes** subtab
3. Click **➕ Create New Recipe**
4. Enter recipe name (e.g., "Spaghetti Carbonara")
5. Enter ingredients (comma-separated)
6. Click **Save**

✅ Recipe is now saved!

### 🛒 Create a Shopping List

**From One Recipe:**
1. Go to **Recipes** tab
2. Find your recipe
3. Click **🛒 Add to Shopping List**

**From Multiple Recipes:**
1. Go to **Shopping Lists** tab
2. Click **🛒 Create New Shopping List**
3. Enter recipe IDs: `recipe-12345, recipe-67890`
4. Click **Create**

✅ Shopping list automatically consolidates all ingredients!

### 📊 Check Pantry Status

1. Click **Pantry** tab
2. See all your ingredients organized by category
3. Quantities show how much you have

### ⚠️ Find Missing Spices (Important!)

This is the **key pantry feature** you requested!

1. Go to **Pantry** tab
2. Click **⚠️ Missing Recommended** button
3. See ONLY the spices you need to restock
4. Click back to **📦 All Items** to see everything

**Example:**
- You have: garlic powder, paprika, oregano
- Missing from recommended: curry powder, cumin, turmeric
- Filter shows only what you need to buy!

### ➕ Add Item to Pantry

1. Go to **Pantry** tab
2. Click **➕ Add Item**
3. Enter:
   - **Item name:** "garlic powder"
   - **Category:** "spices"
   - **Quantity:** "1"
   - **Unit:** "tsp" (or leave blank)
4. Click **Add**

✅ Item appears in your pantry!

### ✓ Mark Item as Used

1. Go to **Pantry** tab
2. Find the item
3. Click **✓ Used** button
4. Quantity decreases

*When quantity reaches 0, item shows "Out of stock"*

## Tips for Success

1. **Meal Prep:** Create shopping lists for your weekly meals
2. **Stock Pantry:** Add all current items to track inventory
3. **Monitor Spices:** Use "Missing Recommended" filter weekly
4. **Update Inventory:** Mark items "Used" as you cook
5. **Restock:** Use missing items list for shopping trips

## Common Workflows

### Workflow 1: Weekly Meal Prep
1. Create 3 recipes for the week
2. Generate shopping list for all 3
3. Go shopping with consolidated list
4. Update pantry as you add items

### Workflow 2: Maintain Spice Rack
1. Go to Pantry → ⚠️ Missing Recommended
2. See which spices you're low on
3. Add them to shopping list
4. Buy and add to pantry

### Workflow 3: Track Pantry
1. Audit current pantry items
2. Add each to Pantry using ➕ button
3. Mark as used when cooking
4. Keep up-to-date inventory
